# 解决GitHub无法显示图片的问题

### 以管理员身份运行cmd

1. win+R

2. 输入cmd

3. ctrl+shift+enter以管理员身份运行cmd

   

### cmd中执行notepad C:\Windows\System32\drivers\etc\hosts以记事本打开hosts文件
